Der Login funktioniert!

Beginne nun damit eigene Dateien, darunter eine index.html Datei, im Verzeichnis private/admin zu erstellen.

Dateien die sich im Ordner private/admin befinden können nur von angemeldeten Benutzern mit Administratorrechten angezeigt werden.

Um einem Benutzer Administratorrechte zu geben muss in seinem Profil die Gruppe admin erhalten sein.

Verlinke gegebenenfalls auf die Seite für den Logout beziehungsweise die Startseite für angemeldete Benutzer.
